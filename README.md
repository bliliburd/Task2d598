## Overview
This repository contains a Python program for **D598 Task 2** that performs the data analysis described in **Task 1**. The script loads the provided Excel dataset and generates an Excel workbook containing the required outputs (duplicates check, state-level descriptive statistics, filtering results, ratio calculations, and a combined dataset).

## Files Included
- **`task2_analysis.py`** — Python script that performs the required analysis and exports results to an Excel workbook.  
- **`README.md`** — Setup and run instructions (this file).  
- **`D598_Task2_Output.xlsx`** — Example output workbook generated by running the script (your run may regenerate this file).

> **Input dataset required (provided by WGU):**  
> `D598 Data Set.xlsx` must be in the **same folder** as `task2_analysis.py` when you run the program.
